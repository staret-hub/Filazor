# Filazor
Blazor File Manager

Target framework
- ASP.NET Core 3.1
- Bootstrap v4.3.1

Icons
- Material Icons
- Open Iconic

Features
- Explore File System
- File Download / Upload
- Login (Default ID/Password: admin/Admin1234!)
